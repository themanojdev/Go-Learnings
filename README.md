## 📚 Learning Resources

Each concept folder contains:
- `.md` files - Detailed explanations and memory points
- `example-code/` - Working Go programs demonstrating the concept
- Comments in code - Explanations of what's happening

---

## 🚀 How to Run Examples

```bash
# Navigate to example-code folder
cd concepts/example-code

# Run any Go file
go run simple-import.go
go run multiple-imports.go
go run custom-imports.go
```

---

## 📝 How to Use This Repository

1. **Read concepts** - Start with `.md` files in `concepts/` folder
2. **Study examples** - Review code in `example-code/` folder
3. **Run code** - Execute examples with `go run`
4. **Practice daily** - Try daily challenges in `daily-progress/`

---

## 🔧 Setup Instructions

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/go-learning.git

# Navigate to project
cd go-learning

# Run any example
go run concepts/example-code/simple-import.go
```
